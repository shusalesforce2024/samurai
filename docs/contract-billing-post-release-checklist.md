## 問題発生時の初動
1. 影響範囲を営業、経理、Freee連携、バッチ、レポートに分類する。
2. Freee連携系は `Freee_Sync_Log__c`、請求、請求明細、商品マスタ、Freee設定を確認する。
3. バッチ問題はスケジュール停止を優先する。
4. データは削除せず、取消・無効化・メモで履歴を残す。
