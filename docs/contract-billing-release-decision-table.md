# 契約請求・Freee連携 リリース判定表

更新日: 2026-05-24

## 判定基準
| 判定 | 条件 |
| --- | --- |
| リリース可 | dry-run成功、RunLocalTests成功、UAT主要シナリオOK、Freee設定確定、商品マスタ投入準備OK、重大な未解決課題なし |
| 条件付き可 | 軽微な運用回避可能課題のみ残る。影響範囲、回避策、対応期限、承認者が明記されている |
| リリース不可 | deploy失敗、Apexテスト失敗、請求金額・Freee連携・権限に重大不具合、または商品マスタ/Freee設定が未確定 |
